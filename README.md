# School Application System

A PHP and MySQL school application system for managing applicant records.

This project uses PHP for the backend, MySQL for the database, phpMyAdmin for database management, and XAMPP as the local development server.

## Tools Used

- PHP
- MySQL
- phpMyAdmin
- XAMPP
- GitHub
